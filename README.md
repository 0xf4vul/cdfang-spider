> 成都房协网数据分析

每隔 15 分钟爬取房协网数据，前端处理元数据，用可视化图标框架渲染。

## 使用

```shell
git clone https://github.com/yhlben/cdfang-spider.git
cd cdfang-spider
yarn
yarn run dev
```

## 特点

- 手动搭建 react 环境
- 手动搭建 koa 环境
- nodejs 爬虫
- 可视化数据分析
